Sobre o Projeto

O objetivo desse projeto é diminuir a variância dos preços dos carros vendidos na concessionária da FORD-USA. Utilizaremos a base de dados fornecida pela a empresa referente ao ano de 2020. Ao finalizar o projeto forneceremos ferramentas para que a equipe comercial informe as caracteristicas do carro e receba qual o preço que deverá ser cobrado. Com essa ferramenta a diretoria terá a certeza de que o valor do carro vendido sempre seja o maior possível, ampliando a margem de lucro e otimizando o tempo da equipe de vendas. A partir de uma base do Kaggle apliquei técninas para análise de dados e de machine learning. Para essa atividade utilizei:

  • Python
  • Pandas
  • Pipeline
  • Xgboost
  • Flask
  • Enconding
  • Scikit-learn
  • Seaborn
  • Matplotlib
  • AWS App Runner
  • ONNX
  • Joblib
  • HTML
  • Github
